# FluxGrab

Небольшое desktop-приложение на Python для получения информации о видео и скачивания в лучшем доступном качестве по ссылке.

Приложение рассчитано только на скачивание контента, на который у пользователя есть права или разрешение.

## Возможности

- вставка ссылки и проверка валидности
- получение названия, длительности, превью и доступных форматов
- выбор режима: `видео + аудио` или `только аудио`
- выбор папки сохранения
- скачивание в отдельном потоке без зависания интерфейса
- прогресс-бар и понятные статусы
- история последних ссылок
- кнопка открытия папки после завершения
- проверка наличия `ffmpeg`

## Стек

- Python 3.11+
- CustomTkinter
- yt-dlp
- Pillow
- requests

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

## Важно про ffmpeg

Для объединения отдельных видео- и аудиопотоков, а также для части аудио-конвертаций нужен `ffmpeg`.

Если `ffmpeg` не найден, приложение:

- покажет предупреждение в интерфейсе
- позволит получить информацию о видео
- предупредит перед загрузкой, если выбранный режим зависит от `ffmpeg`

Установить `ffmpeg` можно любым удобным способом и добавить его в `PATH`.

## Примечания

- История ссылок хранится в домашней директории пользователя: `.fluxgrab/history.json`
- По умолчанию приложение выбирает лучший доступный формат
- Скачивание не стартует автоматически после получения информации
